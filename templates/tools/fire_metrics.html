FBI Table 8 Crime Workbook Upload

Use this when the FBI publishes a new annual city-level crime workbook. Uploading the file only replaces the stored source workbook; the crime scores update the next time you run Refresh All Data.

Open FBI Crime Data Explorer
  1. Click "Documents & Downloads" in the top navigation menu.
  2. Scroll down until you see the section titled "Crime in the United States Annual Reports."
  3. In the first dropdown, select the most recent year available. Note: the newest year listed may not have this specific file ready yet — if downloading gives you an error or an empty/wrong file, try the previous year instead.
  4. In the second dropdown, select "Offenses Known to Law Enforcement." Do NOT select "Law Enforcement Employees Data" — that's a different dataset (police staffing numbers, not crime data).
  5. Click Download. This downloads a .zip file, not a ready-to-upload file yet.
  6. Extract the zip file — right-click it → "Extract All."
  7. Open the extracted folder. You'll see many files (Table 8, 9, 10, 11, etc.) — find and select only the one named starting with "CIUS_Table_8_Offenses_Known_to_Law_Enforcement..."
  8. Come back here, choose that file, and click "Upload Crime Workbook."
Don't rename the file, and don't open it and re-save it in Excel first — re-saving can silently change formatting the import depends on.
